# Agent Diary

Static HTML visualizer for a Paperclip company's recent activity.

## One-command run

```bash
npm install
npm run diary -- --company BUI --days 7
open index.html
```

Or serve over HTTP (required if your browser blocks `file://` fetch):

```bash
cd artifacts/agent-diary && python3 -m http.server 8080
# open http://localhost:8080
```

## Files

- `fetch.ts` — Node script that calls the Paperclip API and writes `data.json`.
- `index.html` — Single-file visualizer (inline CSS + JS). Loads `data.json` or falls back to `data.js`.
- `data.json` / `data.js` — Generated by the fetcher; not committed.
